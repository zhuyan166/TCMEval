### 项目介绍
TCM Evaluation(TCM-Eval)是由具备中医药理论、中医药信息专业背景团队构建的一个中医测评基准，包含了中医药领域如基础研究、临床研究、信息学、政策与管理等方面基础或复杂测评集，可全面评估中医药领域的研究成果、技术水平等方面表现，也可做为中医药领域教育的测试标准，为中医药行业的发展提供有力的支持。
